# 🌐 Projet 1 – Création du site internet Tonfoni

## Aperçu du site

![Site Tonfoni](nouvelle-vaglio.png)

## 📌 Contexte

Lors de mon stage au sein de l'entreprise **Tonfoni**, j'ai participé à la création d'un nouveau site internet.

L'entreprise ne disposait plus de contact avec la personne ayant réalisé son ancien site internet et aucune personne en interne ne possédait les connaissances nécessaires pour en créer un nouveau.

J'ai donc pris l'initiative de me former de manière autonome afin de comprendre toutes les étapes nécessaires à la création d'un site web et de mener le projet de A à Z.

## 🛠️ Outils utilisés

- WordPress
- IONOS
- Canva

## 💡 Réalisations

- Analyse des besoins de l'entreprise
- Réflexion sur l'organisation du site
- Autoformation complète sur WordPress
- Création des différentes pages
- Intégration des contenus
- Personnalisation du design
- Création de certains éléments graphiques avec Canva
- Gestion de l'hébergement avec IONOS
- Mise en ligne du site internet

## 🎯 Compétences développées

- Gestion de projet digital
- Autonomie
- Apprentissage d'un nouvel outil
- Résolution de problèmes

# 🛍️ Projet 3 – Création d'un site e-commerce "NAYAA"

## Présentation du projet

📄 [Voir la présentation du site marchand NAYAA](site-marchand-nayaa.pdf)

## 📌 Contexte

Dans le cadre d'une **SAE de troisième année de BUT Techniques de Commercialisation**, nous avons eu pour objectif de créer un site internet marchand.

Ce projet a été réalisé dans un contexte où nous avons dû être particulièrement autonomes. J'ai donc choisi de repartir de mes connaissances sur WordPress afin de développer un site e-commerce complet.

J'ai approfondi mes connaissances en recherchant par moi-même les solutions permettant de transformer un site classique en véritable boutique en ligne.

## 🛠️ Outils utilisés

- WordPress
- WooCommerce
- IONOS
- Canva

## 💡 Réalisations

- Création de l'identité du projet
- Réflexion sur l'expérience utilisateur
- Création complète du site e-commerce
- Conception des différentes pages
- Création des fiches produits
- Mise en place d'un catalogue produits
- Création du panier
- Configuration du parcours utilisateur
- Création de visuels avec Canva
- Gestion de l'hébergement

Le paiement n'a pas été activé car le projet était réalisé dans un cadre pédagogique et n'avait pas vocation à être commercialisé officiellement.

## 🎯 Compétences développées

- Création d'un site e-commerce
- WordPress
- WooCommerce
- UX/UI
- Gestion de projet digital
- Recherche autonome de solutions
